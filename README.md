# JIdeaLLM (Beta)

**A visual node-based workflow editor for LLM interactions**

> For documentation and detailed setup, see [src/README.md](src/README.md)

## Quick Start

```bash
cd src
npm install
npm start
```

Then open `http://localhost:3000`

## Features

- 🎨 Visual node-based flow editor (ComfyUI/n8n style)
- 🤖 Multi-provider LLM support (Anthropic, Ollama, MiniMax, Poe)
- 🧠 Memory system with RAG-like search
- 📁 Windows content management
- 📝 Template management
- 💬 Chat interface with case management

## Tech Stack

- Node.js + Express
- WebSocket for real-time updates
- Vanilla JS frontend (no framework dependencies)
- JSON file persistence
