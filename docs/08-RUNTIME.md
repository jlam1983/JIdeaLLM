## Workflow

```
┌─────────────────────────────────────────────────────────────────┐
│                    Runtime Workflow                               │
├─────────────────────────────────────────────────────────────────┤
│                                                                  │
│  1. Select Flow                                                  │
│     └─► Load flow definition into runtime                       │
│                                                                  │
│  2. Select/Create Case                                           │
│     └─► Load existing case OR create new case                    │
│                                                                  │
│  3. Chat with LLM                                               │
│     └─► Messages flow through selected flow                     │
│     └─► LLM Worker processes via queue                          │
│                                                                  │
│  4. Export Results                                               │
│     └─► Summary / Snapshot / Chat History                       │
│                                                                  │
└─────────────────────────────────────────────────────────────────┘
```

## Export Options

| Type | Description |
|------|-------------|
| **Export Chat** | Full conversation transcript |
| **Export Summary** | Condensed version |
| **Export Snapshot** | Point-in-time state with all metadata |

## Case Properties

| Property | Description |
|----------|-------------|
| Case ID | Auto-generated unique identifier |
| Case Name | User-defined name |
| Created Date | Auto-recorded |
| Modified Date | Auto-updated on save |
| Associated Flow | Which flow this case uses |
| Initial Data | Starting parameters |
| Chat History | All messages in conversation |
